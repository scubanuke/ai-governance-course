# Module 1 — What We're Actually Governing

## Learning Objectives

After this module, a student will be able to:

- Distinguish generative, agentic, and autonomous AI, and explain why conflating them corrupts governance reasoning.
- Locate the boundary where an autonomous system crosses into safety-critical operation.
- Recognize that boundary as the governance-relevant line the rest of the course targets.


The accessible on-ramp. Sorts generative, agentic, and autonomous AI so the rest of the course has clean vocabulary. **Physically split** so the dated examples can be refreshed without touching the durable concept (see `CONVENTIONS.md`).

## Units, in order

1. `00-concept-autonomy-boundary.md` — **stable** conceptual unit
2. `90-snapshot-2026-examples.md` — **dated** snapshot unit (refresh on a cycle)
3. `in-your-sector.md` — closing reflection
