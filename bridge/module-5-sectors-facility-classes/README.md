# Module 5 — Sectors, Applicability, and Facility Classes

## Learning Objectives

After this module, a student will be able to:

- Define a facility class and the strategic-significance boundary that justifies carving one.
- Apply the four-filter applicability gate to decide whether an instrument applies to a facility.
- Map their own sector onto the method, using the energy-sector carve-outs as models.
- Reason about a genuinely open architectural question (OI-O5 / OI-G9) without a supplied answer.


The bridge. Where your chosen sector stops being cosmetic. You learn what a facility class is, how one gets carved, and then map your own sector onto the method — using the energy-sector carve-outs as models.
